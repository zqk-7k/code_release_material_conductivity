# Data Folder

Place input data here after obtaining it from the corresponding author.

Suggested layout:

```text
data/
├── processed/      # curated conductivity records and database exports
├── model_ready/    # feature matrices, labels and split files
└── dft/            # DFT/CHGNet support tables when available
```

Raw in-house impedance spectra and additional experimental metadata are not bundled in this code release.
